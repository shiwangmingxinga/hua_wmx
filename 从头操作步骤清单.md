# GitHub 同步完整步骤清单

## 📋 第一部分：GitHub 网站操作

### 步骤1：创建 GitHub 仓库
1. 打开 https://github.com
2. 点击右上角 **+** → **New repository**
3. 填写：
   - Repository name: `hua_wmx`（仓库名）
   - Description: `我的项目备份`（描述，可不填）
   - 选择 **Private**（私有）或 **Public**（公开）
4. 点击 **Create repository**
5. 仓库地址：`https://github.com/shiwangmingxinga/hua_wmx.git`

### 步骤2：生成 Access Token
1. 打开 https://github.com/settings/tokens
2. 点击 **Generate new token** → **Generate new token (classic)**
3. 填写：
   - Note: `wmx_hua_backup`
   - Expiration: `30 days` 或 `No expiration`
   - 勾选 `repo`（全部勾选）
4. 点击 **Generate token**
5. **立即复制保存**（只会显示一次！）

## ⚠️ 常见问题

| 问题 | 解决方法 |
|------|----------|
| `error: remote origin already exists` | 远程仓库已关联，跳过步骤6 |
| `Empty reply from server` | 网络问题，稍后重试 `git push` |
| 要求输入密码 | 输入 Token，不是GitHub密码 |
| `nothing to commit` | 代码没有修改，不需要提交 |
